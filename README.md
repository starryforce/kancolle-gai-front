# kancolle-gai
a website to store modified kancolle characters
***
## 前端
> Vue 全家桶、Webpack 打包、UI 组件库 Element
## 后端
> nodeJS、Web 框架 Koa2、ORM 框架 Sequelize、数据库 mySQL
## 编码
> ES6 语法、babel 转码、样式 SASS、兼容到 IE9、RESTful API。
## 编码规范
> Vue 官方风格指南，airbnb eslint 规则
## 测试框架
> mocha + chai
## 文档编写
> MarkDown
## 代码管理
> Github
## 反向代理
> Nginx
## session保存
> Redis
